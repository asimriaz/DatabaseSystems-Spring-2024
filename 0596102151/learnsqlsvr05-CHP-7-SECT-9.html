7.9. Summary

In this chapter, we discussed the set operators available in SQL Server . After reading this chapter, you should have an appreciation of how and when to use UNIONs and INTERSECTs, and how to handle the difference problem, although SQL Server does not have an explicit MINUS operator. Oftentimes queries can be approached in more than one way. In several places, we also showed how the same queries could also be approached without the use of set operators.
