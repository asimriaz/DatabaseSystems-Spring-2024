1.18. Summary

In this chapter, we have shown you how to start Microsoft SQL Server and SQL Server's Management Studio. We have also shown you how to create the Student_course database that we will be using throughout the rest of this book. In addition, we have demonstrated how to work with tables. We have shown you how to type, parse, execute and save a simple query. In the process, we have also familiarized you with the main screens and workings of SQL Server's Management Studio. Towards the end of the chapter, we showed you how to change (or customize) some of SQL Server's default settings to suit your needs.
